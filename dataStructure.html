Sorting Algorithm, BigO & Binary Search

  • Sort Algorithm note on M1
  • Sort Examples & BigO:
    1. Selection Sort | N^2 | In outer loop, declear miniIndex to track the mininal value. If it changed by inside loop, swap the value at the end of outer loop with miniIndex before outloop continue
    2. Insertion Sort | Opposite from Selectoin sort. Outer for loop start from 1. Inside while loop manage the compare backward and element's swap
    3. Shell Sort | n(log(n)^2 |) | use shell original sequence or knuth increments squence to sort elements first, then use same insertion sort algorithms.
    4. Merge Sort(recursive) | n*log(n) | create findMini and merge utility function. split the array into half. base case is array length equal 1. return the merge function with mergSort(firstHalf) and mergeSort(2ndhalf) as argument.
    5. Quick Sort | N^2 | pick a element as pivot point. elements less than that goes to left array, greater goes to right away. return the recurive function with left argument concate with pivot and recursive function with right as argument
    6. Bubble Sort | N^2 | Outer while loop control the passes. Inner for loop control the compare and swap. lastModifiedIndex value to end the while loop
    7. Radix Sort | O(n+k) |(user Matho.pow() to get the single digit. Match single digit to the correct bucket array element. At the end of the loop, concat the spread operator with array argument)
    8. Heap Sort | nlog(n) | convert array into binary heap structure, convert the heap into a max heap, rearrange the array accordling. swap the root node and last node, rearrange the array accordingly. remove the last node from heap. Repeat the process until only root node left.
    9. Cycle Sort(wip)
